# WebGL-Coursework
A JavaScript model with rendering and lighting features, second year university project
Controls:

Mesh controls:
x - Toggle x
y - Toggle y
z - Toggle z
f - Faces rendering mode
e - Edges rendering mode
v - Vertices rendering mode
            
Visiblity controls
c - Toggle cube visiblity
b - Toggle bunny visiblity
		
Lighting controls
1 - Toggle light 1
2 - Toggle light 2
3 - Toggle light 3
4 - Toggle light 4        
5 - Toggle all lights
6 - Turn off lights					

Camera translation controls
up arrow - Up
down arrow - Down
left arrow - Left
right arrow - Right
enter - Zoom in
backspace - Zoom out

Camera orbit controls
w - Orbit up
s - Orbit down
d - Orbit left
a - Orbit right
